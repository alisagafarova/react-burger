### ‚ö° Features

- Fetching ingredient data via API
- Ingredient card modal with photo preview
- Scrollable list of ingredients and smooth navigation to sections
- Drag-and-drop functionality in BurgerConstructor
- Order creation with unique order number
- Total price calculation for burgers
- Nested ingredient sorting
- User authentication and registration
- Order feed and order history pages
- Middleware for handling multiple WebSocket connections
- Component prop validation using PropTypes
- Business logic tested with Jest and functional testing via Cypress
